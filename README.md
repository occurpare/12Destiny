# 🎲 12 DICE - 156 Events Edition

5턴 안에 12칸에 도달하세요! 156개의 다양한 이벤트가 당신을 기다립니다.

## 🎮 게임 방법

1. **주사위 굴리기** - 매 턴 주사위를 굴립니다
2. **이벤트 발생** - 다양한 이벤트가 발생합니다
3. **12칸 도달** - 5턴 안에 골에 도달하면 승리!
